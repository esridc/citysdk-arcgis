CitySDK module for ArcGIS

CitySDK-ArcGIS is a simple JavaScript library for connecting to API geoservices. These services are available from local cities and governments around the world. CitySDK is a JavaScript library developed by the US Census for web developers to quickly connect to the rich sets of national data.
